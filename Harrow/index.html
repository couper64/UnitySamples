Unity Lightweight Shader Graph Example

The tools used to complete this project were Unity 2018.2.6f1 and Visual Studio 2017.

The video above shows an implmentation of Fresnel Effect Shader, Dissolve Effect Shader and Hologram Effect Shader in the first scene.

The second scene demonstrates Water Effect Shader and usage of Topplanar Shader.
Topplanar Shader is a variation of Triplanar Shader utilising only top projection.

Total time spent on project was 2 weeks.
